# jquery.imageTextOverlay.js

> お勉強プラグインシリーズ。画像の上に文字がスライドインするのを簡単にするプラグイン。

## Demo

準備中


## Usage

    // Format:
    $(selector).textOverlay({
      backgroundColor: '000000',
      opacity: '.7',
      color: 'ffffff',
      direction: 'toT',
      tag: 'div',
      className: '',
      transition: true,
      duration: 300,
      easing: 'ease'
    });

    // Examples:
    $('.rollover').textOverlay();

    // Examples:
    $('.rollover').textOverlay({
      backgroundColor: '444444',
      opacity: '.9',
      className: 'overlayText',
    });
